## :information_source: Development setup



```bash
# Clone this repository
$ git clone https://github.com/mailsongarcia/react-github-repo-list

# Go into the repository
$ cd react-github-repo-list

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
